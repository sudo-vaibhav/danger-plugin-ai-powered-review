> use LLMs to find code smells and common anti-patterns before humans review PR

## Usage

Install:

```sh
yarn add danger-plugin-ai-powered-review --dev
```

At a glance:

```js
// dangerfile.js
import aiPoweredReview from 'danger-plugin-ai-powered-review'

aiPoweredReview()
```
